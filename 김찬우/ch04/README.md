# Chapter 4

## 툴
- JPS : 가상 머신 프로세스 상태 도구
- JSTAT : 가상 머신 통계 모니터링 도구
- jinfo : 가상 머신 구성 정보 도구
- JMAP : 자바 메모리 매핑 도구
- JHAT : 힙 덤프 분석 도구
- JSTACK : 스택 추적 도구

## GUI 도구
- JHSDB : 서비스 에이전트 기반 디버깅 도구
- JConsole: 자바 모니터링 및 관리 콘솔
- VisualVM : 다용도 문제 대응 도구
- JMC : 지속 가능한 온라인 모니터링 도구

## 핫스팟 가상 머신 도구
- HSDIS : 디스어셈블러
- JITWatch : JIT 컴파일러 로그 분석 및 시각화 도구
