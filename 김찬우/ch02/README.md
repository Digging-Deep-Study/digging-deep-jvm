  - 스택 프레임: 메서드 호출 시 생성되는 스택의 단위
- 메서드 스택
  - 네이티브 메서드를 호출할 때 사용됨.
- 힙: 객체 인스턴스를 저장. 모든 스레드가 공유.

- 다이렉트 메모리: 네이티브 메모리를 사용하는 자바 객체 저장.
